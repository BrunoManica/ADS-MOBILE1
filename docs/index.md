# ADS Mobile

Bem-vindo às aulas práticas de desenvolvimento mobile com Ionic e React.

Aqui vamos construir, aos poucos, um app de comanda. A ideia é simples: começar com telas pequenas, entender o que cada arquivo faz e evoluir o projeto até ele trabalhar com produtos, pedidos, quantidades, observações e integração com API.

O foco não é decorar código. O foco é entender como um app mobile é organizado por dentro: telas, rotas, componentes, tipos, services e dados fluindo até a interface.

Siga as aulas em ordem. Cada uma adiciona uma parte funcional ao projeto e prepara o caminho para a próxima.
